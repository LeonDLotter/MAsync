Interactive citation network:

Lotter LD, Kohl SH, Gerloff C, Bell L, Niephaus A, Kruppa JA, Dukart J, Schulte-Rüther M, Reindl V, & Konrad K:
Revealing the Neurobiology Underlying Interpersonal Neural Synchronization with Multimodal Data Fusion

Show legend

Description: The network contains all publications included in our meta-analysis of functional magnetic resonance imaging (fMRI) and functional near-infrared spectroscopy (fNIRS) experiments assessing interpersonal neural synchrony (INS) during interaction. Each node represents one publication, edges indicate "citations" (A→B is to be interpreted as "study A cites study B"). Links between studies are build from references and citation lists obtained from OpenCitations data. The OpenCitations database contains only references and citations if these have been made openly available by the publishers. Thus, the edges in this network are not all-encompassing. Node sizes are based on citation-counts within the network. Brains plotted in each node show the brain locations where INS has been detected in the respective publication. For fNIRS publications (blue nodes), grey markers additionally indicate channel positions and thus the brain coverage.

Usage: You can move around the network (and individual nodes) with your mouse or finger or the arrow buttons. You can zoom in with the "+" and out with the "-" buttons. Hovering over a node will show a window with a link to each publication, information on each "experiment" (= data from a unique set of subjects) associated with the publication, and citation/reference counts. Citation/reference counts are provided for the displayed network (NW) and the overall OpenCitations database (OC). The script is optimised for use with large screens and may lead to problems on mobile devices.

Abbreviations: fMRI = functional magnetic resonance imaging, fNIRS = functional near-infrared spectroscopy, INS = interpersonal neural synchrony, ISC = inter-subject correlation, JA = joint attention, JAct = joint action, subj = subjects, y = years.
